# Quantum ESPRESSO Toolchain

This directory contains platform-specific build scripts and notes for
**Quantum ESPRESSO (QE)** used in the QMatSuite ecosystem.

Subdirectories:

- `linux/`   – notes and scripts for building QE on Linux
- `macos/`   – notes and scripts for building QE on macOS
- `windows/` – notes and scripts for building QE on Windows
